# 不可自动化清单

> Claude Code 多 Agent 流水线**无论多智能都做不了**的事情。
> 这些事必须由你（或律师 / 商务 / 运营）介入。
> 主 Agent 跑到这些任务时会**输出阻塞清单 + 暂停（status=stopped）**，等你 `/继续`。

---

## 5 大类阻塞

### 类 1：账户实名

需要**真人 + 身份证 + 信用卡 + 电话号码**的账户注册：

| 服务 | 用途 | 备注 |
|---|---|---|
| 阿里云国际版 | RDS / OSS / CDN / VOD / ECS | 不是国内版！ |
| Cloudflare | WAF / Turnstile / 海外流量套源 | 免费版起步 |
| 域名注册商（Cloudflare Registrar / Namecheap） | 域名 | 不在阿里云国内买 |
| Google Cloud Console | OAuth Client ID（Web + Android 各一份） | 开发者账号实名 |
| Google Play Console | 应用上架 + 内购 | 一次性 25 USD 注册费 + 实名 |
| Twilio / 阿里云国际短信 | 手机 OTP | 必须**预付费**，禁后付费裸奔 |
| Sentry | 错误监控 | 免费版 |
| UptimeRobot | 拨测 | 免费版 |
| Telegram | 告警 channel + bot | @BotFather 拿 token |
| GitHub | 代码仓库 + Actions | 个人账号或组织账号 |
| 1Password | keystore + 凭据备份 | 团队版推荐 |
| Termly / iubenda | 隐私政策 + 服务条款生成 | 海外合规模板 |
| Apple Developer（如未来上 iOS） | App Store 上架 | 99 USD/年 + 实名 |

**主 Agent 跑到 M0「账户与域名」里程碑时**：
1. 输出上表对应的 checklist
2. 流水线 status=stopped + reason=user_action_required
3. 等你买完账户 + 把凭据存 1Password + 写 docs/account-list.md
4. 你敲 `/继续`，主 Agent 询问「M0.1 域名买好了吗？(y/n)」逐项确认

### 类 2：法务 / 合规

需要律师审核或政府报备：

| 项 | 谁做 | 备注 |
|---|---|---|
| 隐私政策审核（GDPR / 印尼 PDP / 巴西 LGPD / 沙特 PDPL） | 律师 + Termly 模板兜底 | 上线前必须做 |
| 服务条款审核 | 同上 | 同上 |
| 内容版权采购合同 | 你 + 商务 + 律师 | 影视版权水深，先小范围采购测试 |
| 印尼 PSE 注册（Kominfo） | 你 + 当地代理 | DAU > 10w 触发，否则不强制 |
| 中东内容审查合规（沙特 / 阿联酋） | 你 + 当地合规顾问 | 看具体内容 |
| 巴西 LGPD 注册 | 同上 | DAU 起来后做 |
| Play / Apple 商店应用审核 | 平台审核员（你提交） | 隐私政策 + 内容评级 + 截图等 |
| 删除账号合规（Play 硬卡） | 你提交隐私政策 + App 内入口 | 已在 user 模块实现 `/users/me/delete-request`，但实际审核要看页面截图 |

**主 Agent 不可代办**：
- 写隐私政策**初稿**可以（Claude 能起草）
- 但**最终审核 + 签字 + 报备**必须律师 + 你
- Claude 写的隐私政策在文档里要明确标注「**初稿，未经律师审核，上线前必须复核**」

### 类 3：商务 / 内容

需要商业谈判或采购决策：

| 项 | 谁做 | 备注 |
|---|---|---|
| 内容版权采购（影视 / 综艺 / 动漫） | 你 + 商务 | 单方面 Claude 不能签合同 |
| Play Billing / 支付通道开通 | 你（Google 实名 + 银行账户） | 上 VIP 时才需要 |
| 广告 SDK 接入（AdMob / Unity Ads） | 你（Google 账号 + 收款账号） | 起步靠广告变现 |
| CDN / VOD 商务谈判（流量上量后） | 你 + 商务 | 起步用阿里云国际默认价 |
| 客服外包决策 | 你 | DAU > 1w 后考虑 |

**Claude 能做的**：
- 写 SDK 接入代码（一旦你给了 SDK key + 配置参数）
- 写广告位埋点 + 后台广告位管理
- 但**不能代你谈合同 / 选 SDK 厂商 / 决定广告策略**

---

### 类 4：运营 / 增长

需要业务判断 + 长期投入：

| 项 | 谁做 | 备注 |
|---|---|---|
| DAU 真实增长（从 1k → 100w） | 运营团队 + 营销预算 + 内容质量 | 不是写代码能交付 |
| 用户调研 / NPS / 留存分析决策 | 运营 / 数据分析 | Claude 能写 SQL 但解读决策是你 |
| A/B 测试设计 / 结果判读 | 数据 + 产品 | Claude 能写 A/B 框架代码 |
| 内容审核团队招募（高敏感地区） | 你 / HR | RAM 子账号 SOP Claude 已写 |
| 客服 / 用户投诉处理 | 你 / 客服外包 | |
| 应用商店运营（关键词 / 截图 / 描述） | 你 / 运营 | Claude 能给文案建议 |

**核心提醒**：**100w DAU 不是技术指标，是商业指标**。
即使代码 100% 完美，没有内容 / 营销 / 运营，DAU 也不会到 100w。
代码是必要不充分条件。

### 类 5：物理 / 现实操作

需要碰真实硬件 / 物理动作：

| 项 | 备注 |
|---|---|
| keystore 三地备份（GPG U 盘） | 红线 #1。Claude 能加密 + 给指令，但你要插 U 盘 |
| 离线密码本 / 凭据物理备份 | 同上 |
| 应用商店审核员要求的视频 demo | 你录屏 |
| 海外节点 TTFB 实测（webpagetest） | Claude 能解读结果但发起测试要你 |

## 主 Agent 行为规范

跑到 `blocks_to_user=true` 里程碑时（manifest.json 标记）：

### 1. 输出阻塞清单
按上表对应里程碑的具体动作，输出可勾选 markdown：

```markdown
## 流水线暂停 — 等你完成 M0「账户与域名」

主 Agent 不能代你做以下事情。请你完成后任意窗口敲 `/继续`：

- [ ] 0.1 在 Cloudflare Registrar 或 Namecheap 买域名（不要在阿里云国内买）
- [ ] 0.2 注册阿里云国际版账号（alibabacloud.com，**不是 aliyun.com**），完成实名 + 支付方式
- [ ] 0.3 注册 Cloudflare 账号
...

**完成后**：
- 把所有凭据存进 1Password（不要进 Git！）
- 写 `docs/account-list.md`（资源清单 + URL，**不写密码**）
- 敲 `/继续` — 主 Agent 会逐项问 (y/n) 确认你完成了哪些，然后继续后续里程碑
```

### 2. 写状态
- `active.json.status = "stopped"`
- `active.json.blocked_reason = "user_action_required"`
- `active.json.blocked_milestone = "<id>"`
- `timeline.log` 写 `BLOCKED_USER_INPUT <milestone-id>`

### 3. 退出
**不要尝试自动绕过**（如：尝试用 selenium 注册 Cloudflare 账号 — 这是禁忌）。
直接 return STOP，等用户 `/继续`。

### 4. 用户 `/继续` 时
- 如果该里程碑还有 `blocks_to_user=true` 任务未完成 → 询问 "M0.1 已做？(y/n)" 逐项
- 用户全 yes 后 → 标 milestone done → push → 进下一里程碑
- 用户某项 no → 该任务保持 pending → 主 Agent 暂停等下次

## 永远不要假装能代办

如果用户说「帮我注册阿里云账号」「帮我提交应用商店审核」「帮我谈版权」：

**回应模板**：

> 这件事需要你本人实名 + 信用卡 / 谈判，Claude Code 多 Agent 流水线无法代办（详见 [`docs/不可自动化清单.md`](docs/不可自动化清单.md) 类 X）。
> 我可以帮你：
> - 起草准备资料 / 申请文案 / 沟通邮件草稿
> - 写完整的注册步骤 SOP（哪个页面点哪个按钮）
> - 你做完后 verify 配置正确（如：测 RDS 连通 / 测 SMS 发送）
> 不能做：替你点"注册"按钮、替你输入身份证、替你签合同。
