# /状态

只读地查看 `.claude/runs/active.json` 指向的 run 的进度，不触发任何执行。

---

## 执行步骤

### Step 1：读 active.json

```bash
cat .claude/runs/active.json 2>/dev/null
```

不存在 → 汇报「无活跃 run」，列出 `.claude/runs/` 下所有历史 run-id 给用户参考。结束。

### Step 2：读所有状态文件

```bash
RUN_ID=<from active.json>
cat .claude/runs/$RUN_ID/manifest.json
cat .claude/runs/$RUN_ID/milestones.md
wc -l .claude/runs/$RUN_ID/tasks_queue.jsonl
wc -l .claude/runs/$RUN_ID/tasks_done.jsonl
wc -l .claude/runs/$RUN_ID/tasks_skipped.jsonl
tail -30 .claude/runs/$RUN_ID/timeline.log
```

### Step 3：汇报模板

```markdown

**标题**：<title>
**起始**：<started_at>（运行 X 小时 Y 分钟）
**状态**：running / stopped / completed / aborted
**最近事件**：<last_event_at>
**最近 commit**：<last_commit> "<message>"

---
```

```
## 操作建议

- 想继续跑 → `/继续`
- 想暂停（关窗口前优雅停） → `/停止`
- 想丢弃重来 → `/丢弃 <run-id>`（**不可逆**）
- 想看 FAIL 任务的详细修复建议 → `cat .claude/runs/<run-id>/tasks_skipped.jsonl | jq`
```
